# VisionPro 仪表板项目

## 项目背景

本项目是为微博内部开发的VisionPro数据可视化仪表板，主要用于展示和分析关键业务指标。项目基于Next.js框架开发，采用React和TypeScript技术栈。

## 主要功能

- 实时数据监控
- 多维数据分析
- 自定义报表生成
- 预警通知系统

## 技术栈

- Next.js 15.3.0
- React 18
- TypeScript 5.0
- Tailwind CSS

## 快速开始

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 访问应用
打开浏览器访问 http://localhost:3000
